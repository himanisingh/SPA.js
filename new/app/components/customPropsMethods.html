Component's Extended Properties

Custom Properties/Methods

A component can have it's custom properties and methods

Accessing Component’s Extended Properties and Methods

The component’s extended properties and methods can be accessed using app.componentX.PropertyOrMethodName or app.componentX[ 'PropertyOrMethodName' ]

Within extended methods, the properties or methods can be accessed using this. keyword.

Note: this.$data refers to the Component’s Final Data (base property) i.e. The final data which goes to the template. (Refer above example)
