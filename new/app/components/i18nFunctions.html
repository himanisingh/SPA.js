Passing Params to Language Text

The i18n text can have varibles to display values in run-time using single curly braces { variableName }

Use data-i18n-data attribute in Component template to pass the values to the variables.

Use spa.i18n.text function to pass the values to the variables.

i18n Functions

spa.i18n.text( i18nKey ) ==> returns the text for the i18nKey from the active language file.

spa.i18n.text( i18nKey, paramsValue ) ==> returns the text for the i18nKey (replaced variables with paramsValues) from the active language file.

spa.i18n.apply() ==> applies active language's i18n text for the entire body elements.

spa.i18n.apply( context ) ==> applies active language's i18n text for the context's elements.

spa.i18n.apply( context, elSelector ) ==> applies active language's i18n text for the filtered elements (elSelector) in context.
