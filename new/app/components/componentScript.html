Component's Script

The Basics

A component Script file componentX.js contains two SPA component helper functions.

spa.$( 'componentX' , { } ) to define Component's base properties like template, target, dataUrl etc.
spa.$extend( 'componentX' , { } ) to define Component's extended properties like events and other custom actions to be performed on component UI.

Refer Component's Base and Extended Properties section for more details.
