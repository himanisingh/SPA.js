Component's Base Properties

dataUrl

Specify actual API path with reference to the index.html

The SPA will attempt to fetch the data from //host/api/path/to/data

All API URLs can be stored in spa.api.urls and refer the key in dataUrl. For more details refer SPA API section.

dataUrl is an optional property. If no data-binding in template, ignore this property.
All base properties can be overridden before the render call using spa.$render( ) method.
